# oled_driveer_for_SSD1306
0.96英寸OLED显示驱动器，带有SSD1306芯片，包含stm32cubeide_hal库的中文注释，易于移植和管理
2024年7月22日
开发硬件为 stm32f103c6t6，似乎在运行bmp的时候有点不够用
内部函数大部分都有中文注释，建议改个cubeide的字体再使用
开发使用的屏幕是0.96寸的oled，芯片是SSD1306，使用的是IIC通信进行测试
